# 외부 구인 채널 자동 소싱 실현성 리서치 (2026-07-22)

> 딥리서치(6각 검색 · 소스 22개 · 주장 101→검증 25→**확정 19**, 3표 적대 검증) 결과를 종합.
> 질문: 시니어(50~70대) 긱워커·배달/물류 인력을 외부 채널에서 **합법·자동화 가능한 유료 경로**(유료 게시/광고·공식 API·대행사)로 모을 수 있는가? 채널별 go/no-go + 파일럿.
> ⚠️ 세션 한도로 종합 단계만 미완 → 검증 통과 주장 기반으로 수기 종합. 일부 항목은 미검증(하단 명시).

## 결론
**"에이전트 무인 자동 게시/DM"은 대부분 정책상 막혀 있거나 파트너 게이트다. 현실 경로는 유료 광고 + 반자동(에이전트가 초안·소재·타겟 세팅 → 사람/대행이 집행).**

## 채널별 판정

| 채널 | 유료 경로 | 합법 자동화 | 시니어 타겟 | 판정 |
|---|---|---|---|---|
| 오카방(오픈카톡) | 광고상품 불명확 | 봇 게시·DM **정책 정면 위반**, 대량 제재 | — | 🔴 NO-GO |
| **밴드(BAND)** | GFA 피드/스마트채널 | **셀프서브·대행 가능**(API는 파트너사) | **5세 단위(50·60대)** ✅ | 🟢 GO (1순위) |
| 알바몬 | 배달·물류 **유료만** (3,300~231,000원/일, 즉시등록 15,400원/건) | 공개 게시 API 미확인 → 반자동/대행 | 직군 도달 강함 | 🟢 GO (반자동) |
| 네이버 GFA | 성과형 디스플레이 | REST API 有, **파트너사만** → 대행 | 지역·관심사 | 🟢 GO (대행) |
| 당근 | **당근알바 광고**(공식, 최소 5,000원/일, 동 단위) | **API·자동 없음**(수동) | 연령 타겟 **불가**(고용차별법) | 🟡 조건부(수동) |
| 메타/인스타 | 마케팅 API | 자동 집행 가능하나 **채용광고 연령 18–65+ 잠금**(시니어 정밀 불가; 한국 적용 불명) | 약함 | 🟡 조건부 |
| 잡코리아 | 유료 게시 | 공개 게시 API 경로 **불명확**(검증 미확정) | — | 🟡 확인 필요 |

## 핵심 시사점
1. **오카방 자동화는 접지 말 것** — 봇/매크로 게시·DM·다중계정 반복발송이 카카오 정책 정면 위반. 2025 상반기 오픈챗에서 스팸·운영방해로 82,547계정 이용제한(실 집행 중).
2. **완전 무인 에이전트 집행은 대부분 막힘/파트너 게이트.** 합법 자동화 = 광고 API(네이버·밴드), 파트너사 경유 또는 셀프서브 수동 세팅.
3. **시니어 정밀 연령 타겟이 되는 유료 채널은 사실상 밴드(BAND)뿐** — 메타는 채용광고 연령 잠금, 당근은 연령 타겟 불가.
4. **당근 정공법 = '당근알바 광고'**(공식 유료, 동네 기반). 과거 문제였던 '당근 링크 유도'와 별개. → 공고탭의 당근 게시 링크(source=danggeun)는 정공법이 아님.

## 파일럿 추천
- **1순위 — 밴드 피드광고 소액 테스트**: 시니어 도달 1위 SNS + 5세단위 연령·지역 타겟. 옹보딩은 소재·타겟·랜딩(`/apply?source=…`)만 세팅. 예산 일 1~3만원×2주, 성공지표 = 리드당비용(현 SMS 재컨택 ~530원 대비)·지원 전환율.
- **2순위 — 알바몬 유료 게시(반자동)**: 배달·물류 정공법. 에이전트 채널별 초안 생성 → 사람 즉시등록. ← 아래 주제 D 재정의의 실사용 근거.
- 병행 — 당근알바 광고 동네 소액 테스트(수동).

## 로드맵 반영
- **주제 D(게재 흐름) 재정의 = "반자동 초안 인프라"**: D1(채널별 본문 저장)·D2(화주사·지점 ai_facts→초안 생성 연계)은 밴드/알바몬 반자동 집행을 돕는 인프라로 **유효**.
- **D4(당근 자동 게시 링크)는 폐기** — 정공법(당근알바 광고, 수동)과 다르고 과거 정책위반 소지.
- D3(모집방식 정합)은 소소한 정리로 D1과 함께.

## 검증 한계 (집행 전 직접 확인 필요)
- **미확정(세션 한도로 표결 실패):** 카카오모먼트 광고의 오픈챗 지면 노출 가능성 · 잡코리아 API 게시 가능성/공공기관 한정 여부 · 메타 채용광고 연령잠금의 한국 적용.
- **기각된 주장(사실로 쓰지 말 것):** "잡코리아 API=조회 전용, 게시 불가"(3표 기각) · 배달 긱 포화 통계 · 카카오 비정상프로그램 특정 프레이밍.
- 시니어 앱 사용 순위(카톡·네이버 우위, 당근 9위대)는 방향성 근거 수준.
